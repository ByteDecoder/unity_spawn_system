Member Function Documentation

void RoundManager.RoundCheckpoint.Enter ( )

Invoked by Round. Not required to be called manually. Called at the beginning of each checkpoint.

void RoundManager.RoundCheckpoint.Execute ( )

Invoked by Round each time step that checkpoint is active. Not required to be called manually. Raises EnemySpawnRequestEvent if:

  • RoundEnemies count greater than 0.
  • The current time between enemy spawns is >= TimeBetweenEnemySpawns.
  • A random value between 0 and 1 is <= EnemySpawnChance.

float RoundManager.RoundCheckpoint.EnemySpawnChance = 1

A number between 0 and 1. Every time an enemy is to be spawned, a random number between 0 and 1 is generated. If this random number is less than or equal to this then the enemy is spawned. Higher number results in a greater chance to spawn an enemy and 1 results in a spawned enemy every time.

bool RoundManager.RoundCheckpoint.LimitEnemyCount

The number of enemies spawned during this checkpoint will be limited if this is true.
